# 本课程介绍

今年准备重新把所有技术基础重新学习梳理一遍，查缺补漏，扎实提升。年初遇到了极客时间的覃超的《算法训练营》，偏实战；另一个课程就是王争的《数据结构与算法之美》。


感受：超值；争哥的课程更像是简明清晰的书本，超哥的课程更像是课后练习册，二者配合，事半功倍。

知识有价，官方课件恕不提供。本目录下记载自己学习过程中摘录的关键笔记，或称之为知识点搬运，便于自己快速温习和索引。如有人志同道合小伙伴看到，不喜可喷。
